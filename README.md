# Django-Ecommerce
A Python project with Django Framework of a simple Ecommerce WebSite to logged in and Anonymoous Users

### Project Resume and Technologies
- This is a Django Ecommerce project with Paypal Integration; Authenticated users AND Anonymous users can buy from the shop. For the Anonymous users we create/manipulate cookies. We use API for update products for DB
- Django
- Paypal API Integration
- Languages: Python and JavaScript
### Objective
This project has the objective of my learning into Django World. Learning how to manipulate cookies and use the powerfull django library
